VirtualBox Manager

  • Control Oracle VirtualBox® installation remote & secure using Webservice API over SSL
  • Exposes all API functionality, i.e. start/stop, snapshot, save state
  • Developed for latest Android APIs but still backward compatible with Gingerbread.
  • Display Virtual Machine details & log files
  • Real-time CPU/Memory metric graphs
  • 4.5-stars (1,000->5,000 installs) on Google Market
  • Source code available at GitHub

SSL Support

SSL can be enabled when launching `vboxwebsrv` on the server machine. You will need:

  1. server certificate & key inside a PEM file
  2. CA certificate in a CRT file
  3. Launch vboxwebsrv --host 0.0.0.0 --port 18084 --ssl --serverKey server.pem --ca ca.crt
